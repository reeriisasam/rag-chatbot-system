# 🖥️ GUI Version - RAG Chatbot

## การใช้งาน GUI Window (เหมือน OpenCV)

### 🚀 เริ่มต้นใช้งาน

```bash
# รัน GUI version
python main_gui.py
```

### ⌨️ การใช้งาน

#### 1. โหมดข้อความ (Text Mode)
- พิมพ์คำถามในช่อง input
- กด **Enter** หรือคลิก **Send ➤**
- Bot จะตอบกลับในพื้นที่ chat

#### 2. โหมดเสียง (Voice Mode)
- คลิกปุ่ม **🎤 Voice**
- พูดคำถามเข้าไมโครโฟน
- Bot จะตอบกลับทั้งข้อความและเสียง

#### 3. คำสั่งพิเศษ
พิมพ์คำสั่งเหล่านี้ในช่อง input:

- `/help` - แสดงคำสั่งทั้งหมด
- `/clear` - ล้างหน้าจอ
- `/stats` - แสดงสถิติระบบ
- `/reload` - โหลดเอกสารใหม่
- `/mode` - สลับโหมด text/voice

### 🎨 ฟีเจอร์ GUI

#### สีที่ใช้
- **เขียว (#00ff88)** - ข้อความของคุณ
- **ฟ้า (#00bfff)** - ข้อความของ Bot
- **เหลือง (#ffaa00)** - ข้อความระบบ
- **เทาเข้ม (#2d2d2d)** - พื้นหลัง chat
- **ดำ (#1e1e1e)** - พื้นหลังหลัก

#### ปุ่มต่างๆ
1. **Send ➤** - ส่งข้อความ
2. **🎤 Voice** - เปิดโหมดเสียง
3. **🗑️ Clear** - ล้างหน้าจอแชท
4. **📝 Text Mode / 🎤 Voice Mode** - สลับโหมด
5. **📊 Stats** - แสดงสถิติระบบ

#### สถานะ
- **● Online** (เขียว) - พร้อมใช้งาน
- **● Processing...** (เหลือง) - กำลังประมวลผล
- **● Offline** (แดง) - ไม่พร้อมใช้งาน

### 🎯 เคล็ดลับการใช้งาน

1. **Shift + Enter** - ขึ้นบรรทัดใหม่ในช่อง input
2. **Enter** - ส่งข้อความทันที
3. **ESC หรือปิดหน้าต่าง** - ออกจากโปรแกรม
4. **คลิกขวาในพื้นที่ chat** - คัดลอกข้อความ

### 🔧 การตั้งค่า

แก้ไขไฟล์ `config.yaml` เพื่อปรับแต่ง:

```yaml
ui:
  window_width: 900      # ความกว้างหน้าต่าง
  window_height: 700     # ความสูงหน้าต่าง
  theme: "dark"          # ธีม (dark/light)
  font_size: 10          # ขนาดตัวอักษร
```

### 🐛 แก้ปัญหา

#### ปัญหา: หน้าต่างไม่แสดง
```bash
# ตรวจสอบว่ามี tkinter หรือไม่
python -m tkinter
# ควรเห็นหน้าต่างทดสอบ
```

#### ปัญหา: ปุ่ม Voice ไม่ทำงาน
```bash
# ติดตั้ง audio packages
pip install SpeechRecognition pyttsx3 pyaudio
```

#### ปัญหา: ตัวอักษรไม่สวย
- ใช้ Windows 10/11 ที่มี Segoe UI font
- หรือแก้ไขใน `gui_window.py` เปลี่ยน font

### 💡 ข้อดีของ GUI Version

1. **ใช้งานง่าย** - คลิกปุ่มแทนพิมพ์คำสั่ง
2. **สวยงาม** - มี UI ที่ออกแบบมา
3. **สะดวก** - คัดลอกข้อความได้ง่าย
4. **เป็นมิตร** - เหมาะกับผู้ใช้ทั่วไป
